# GAMES.JS

##### Wellcome to games.js, currently you can play these games:

- type **hello** to see full info

- Snake game **UPDATED!**
- Race game
- guessWord game **NEW!**

## Requirements

- node.js
type this before playing!
```bash
npm ci
```
or this if you have make
```bash
make install
```
- make (you dont really need it).

## How can i play ?

##### Choose the game that you want to play, and type it's name to the console.
